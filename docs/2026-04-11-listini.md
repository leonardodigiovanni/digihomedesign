# Listini — griglia prezzi acquisto/vendita per preventivi

**Data:** 2026-04-11  
**Stato:** completato

---

## Obiettivo

Sezione `/listini` (id 25, già in nav-config) con una griglia completa di articoli/lavorazioni.  
Ogni record ha un prezzo di acquisto (costo interno) e un prezzo di vendita (da preventivare al cliente).  
I record sono **modificabili inline** perché i prezzi cambiano nel tempo.

---

## Struttura DB — tabella `listini`

| Colonna           | Tipo                    | Note                                      |
|-------------------|-------------------------|-------------------------------------------|
| `id`              | INT PK AUTO             |                                           |
| `categoria`       | VARCHAR(100) NOT NULL   | Es. "Infissi", "Posa", "Ceramiche"        |
| `produttore`      | VARCHAR(100) NOT NULL DEFAULT '' | Es. "Schüco", "Mapei", "Generico" |
| `descrizione`     | VARCHAR(300) NOT NULL   | Es. "Finestra scorrevole 2 ante"          |
| `unita`           | VARCHAR(30) NOT NULL    | Es. "m²", "kg", "pz", "h", "ml", "t"     |
| `prezzo_acquisto` | DECIMAL(10,2) NOT NULL DEFAULT 0 | Costo interno                  |
| `prezzo_vendita`  | DECIMAL(10,2) NOT NULL DEFAULT 0 | Prezzo al cliente              |
| `note`            | TEXT NULL               | Info aggiuntive                           |
| `created_at`      | TIMESTAMP DEFAULT CURRENT_TIMESTAMP |                               |
| `updated_at`      | TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | |

## UI

### Filtri (barra superiore)
- **Ricerca testo** — filtra su descrizione + produttore
- **Categoria** — select con valori distinti presenti in tabella
- **Produttore** — select con valori distinti presenti in tabella
- Contatore record visibili

### Tabella
Colonne: Categoria · Produttore · Descrizione · Unità · P.Acquisto · P.Vendita · Margine% · Note · Azioni

- **Margine%** calcolato client-side: `((vendita - acquisto) / vendita * 100)` — colorato (verde se > 20%, giallo se 10-20%, rosso se < 10%)
- **Prezzi evidenziati** in colori distinti (acquisto = blu scuro, vendita = verde scuro)

### Modifica inline
- Doppio click su qualsiasi cella del record → riga diventa editabile (tutti i campi)
- Pulsante ✓ Salva e ✕ Annulla appaiono sulla riga
- La data `updated_at` viene aggiornata automaticamente dal DB

### "+ Nuovo articolo"
Form espandibile in testa alla tabella con tutti i campi.  
Unità: select con opzioni predefinite (`m²`, `ml`, `kg`, `t`, `pz`, `h`, `corpo`) + voce libera.

### Elimina
Pulsante per riga, con conferma.

## Accesso

- Id pagina: **25** (già "Listini" in nav-config, href da `/interno/25` → `/listini`)
