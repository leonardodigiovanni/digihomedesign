## Controlli di accesso

### Per-utente (`users.miei_ordini_visibili`)
- Nuova colonna `miei_ordini_visibili TINYINT(1) DEFAULT 1` su `users`
- In `app/layout.tsx`: se cliente e `miei_ordini_visibili = 0` Ôćĺ rimuove id 30 dai rolePermissions (stesso pattern cantieri)
- In `app/gestione-utenti/`: colonna toggle "I Miei Ordini" per utenti cliente
- `nav-config.ts`: admin non vede id 30

### Per-ordine (`ordini.visibile_cliente`)
- Pulsante toggle nella riga staff (­čĹü Vis. / ­čöĺ Nasco.) ÔÇö stesso stile cantieri
- Query cliente: `WHERE o.visibile_cliente = 1`
